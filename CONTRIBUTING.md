# Contributing

Thanks for your interest in contributing to Page Copilot by Alon.

## Before You Start

- Please check existing Issues before opening a new bug report or feature request.
- Use Discussions for usage questions, idea exploration, and open-ended design debate.
- Keep changes scoped. Avoid mixing refactors, feature work, and unrelated cleanup in one PR.

## Development Setup

1. Clone the repository.
2. Open `chrome://extensions/` in Chrome.
3. Enable Developer Mode.
4. Click "Load unpacked" and select this project directory.
5. Reload the extension after code changes.

## Contribution Guidelines

- Prefer small, reviewable patches.
- Preserve existing behavior unless the change is intentional and documented.
- Update documentation when the user-facing behavior changes.
- Do not commit secrets, API keys, or personal credentials.

## Bug Reports

When filing a bug, include:

- Chrome version
- Reproduction steps
- Expected behavior
- Actual behavior
- Screenshots or console logs when relevant

## Pull Requests

For pull requests, please:

- Explain what changed and why
- Describe any user-facing impact
- Mention how you verified the change
- Keep the PR focused on one problem

## Security

Do not publish secrets or real API keys in Issues, Discussions, screenshots, logs, or pull requests.
