# Privacy Policy

## What this extension stores locally

- API configuration such as API URL, selected model, and API key is stored in Chrome local storage on your device.
- Recent page metadata used for cross-page comparison is stored in Chrome session storage on your device. This may include page title, URL, domain, and visit time.
- Recent chat history is stored locally in Chrome local storage on your device for conversation continuity.

## What is sent to external AI services

- When you explicitly ask Page Copilot to summarize, translate, explain, or answer with page context, the relevant selected text, page text, video transcript text, or prompt content is sent to the AI API endpoint you configured.
- The extension does not send page content to an external AI service unless you trigger an AI action.

## What this extension does not collect

- No account system is built into the extension.
- No analytics, advertising trackers, or remote telemetry are included by default.
- No browsing history is uploaded to a developer-owned server by default.

## User control

- You can clear conversation history from the extension UI.
- You can remove or change your configured API endpoint and API key at any time.
- You can remove the extension at any time from `chrome://extensions/`.

## Contact

- For bug reports and feature requests, use the repository Issues after publishing the project.
- For usage questions and open-ended discussion, use the repository Discussions after publishing the project.
